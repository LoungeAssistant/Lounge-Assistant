Welcome to Lounge Assistant.

Lounge Assistant is a GreaseMonkey script that improves the look and the functionnality of csgolounge

It's deeply inspired by Lounge Companion. And it tries to mimic the comportement.

Installation

First you need to install GreaseMonkey by clicking on this link https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/

After that you just need to click on this link, a popup will open asking you if want to install.

Everything is ready to go, you just need to log on csgolounge

Functionality

  • Fancy design
  • Display item price on hover
  • Display item quality
  • Enhanced Item Preview
  • Auto update

It's open source !

Everything is open source, so feel free to contribute on github on this repository
