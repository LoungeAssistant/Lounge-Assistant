Lounge-assistant

The csgolounge.com Addon

Welcome to Lounge Assistant.

Lounge Assistant is a GreaseMonkey / TamperMonkey script that improves the look and the functionality of csgolounge

It's deeply inspired by Lounge Companion, but with a lot more tools !

It's compatible with Chrome and Firefox.

Functionalities

  • Fancy design
  • Try to submit bet every 10s, no more button smashing !
  • Display "Best of #" on inedx page
  • Minimize / Maxizime Match
  • Display item price and quality
  • Custom Background
  • Currency choice
  • Bot Status on every page
  • 'Bump all trade' button on menu
  • Direct link to inventory on user profile
  • Win / Loss ration and bet history
  • Enhanced Item Preview
  • Auto update
